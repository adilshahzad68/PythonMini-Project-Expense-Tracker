# PythonMini-Project-Expense-Tracker

Description

The Expense Tracker is a simple Python command-line application to help users manage and track their daily expenses. It allows users to add, view, and calculate total expenses easily, with all data stored locally in a JSON file (expenses.json).

Features
Add Expense – Record an expense with date, category, description, and amount.
View Expenses – Display all recorded expenses in a clear, organized format.
Total Expense – Calculate and display the total amount spent.
Persistent Storage – All expenses are saved in a local JSON file (expenses.json) for future use.
